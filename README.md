# 🧬 GenomeGPT

GenomeGPT is an AI-powered genomic research assistant that allows users to upload genomic datasets and ask natural language questions about mutations, genes, and DNA sequences.

---

# Features

- Upload FASTA and CSV genome datasets
- AI-powered genomic chatbot
- Gene and mutation explanations
- Beginner-friendly genomic insights
- Streamlit web interface

---

# Tech Stack

- Python
- Streamlit
- OpenAI API
- Biopython
- Pandas
